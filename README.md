# Element-Arena
An Arabic memory card game built with HTML, CSS and Vanilla JavaScript.

# 🃏 ميدان العناصر | Element Arena

An Arabic Memory Card Game built using **HTML**, **CSS**, and **Vanilla JavaScript**.

Element Arena transforms the classic memory card game into a strategic experience by introducing magical abilities, multiple difficulty levels, a shop system, inventory management, sound effects, and a responsive user interface.

- 🃏 Classic Memory Card Gameplay
- ⚡ 7 Unique Magical Abilities
- 🪙 Coin Reward System
- 🛒 In-game Shop
- 🎒 Inventory System
- 🏆 Local Leaderboard
- 🔊 Sound Effects
- ⏳ Countdown Timer
- 📱 Fully Responsive Design
- 💾 Local Storage Save System
